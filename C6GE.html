C6GE

C6GE is my flagship project – a modern, open-source C++ game engine currently using OpenGL. While it currently supports only Windows, I am actively working to expand compatibility to macOS and Linux. The goal is to build a fast, lightweight, and versatile engine for developers targeting multiple platforms.

License: C6GE is proudly licensed under the MIT License.

Current Features

  • Written in modern C++
  • OpenGL rendering pipeline (DirectX, Vulkan, Metal planned)
  • Basic 3D object rendering (cube, sphere, etc.)
  • Real-time rendering preview
  • Input handling and event loop system
  • Basic scene graph and entity management
  • Simple transformation system (translate, rotate, scale)
  • Modular design for portability and extendibility
  • Lightweight build setup

Technical Specifications

  • C++17-based codebase
  • OpenGL for graphics rendering
  • Custom-built windowing and input system (no third-party engine dependencies)
  • Cross-platform architecture (Windows support now, macOS/Linux coming)
  • Future support planned for Vulkan, Metal, and DirectX

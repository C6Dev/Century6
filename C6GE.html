Current Features

  • Written in modern C++
  • OpenGL rendering pipeline (DirectX, Vulkan, Metal planned)
  • Basic 3D object rendering (cube, sphere, etc.)
  • Real-time rendering preview
  • Input handling and event loop system
  • Basic scene graph and entity management
  • Simple transformation system (translate, rotate, scale)
  • Modular design for portability and extendibility
  • Lightweight build setup

Technical Specifications

  • C++17-based codebase
  • OpenGL for graphics rendering
  • Custom-built windowing and input system (no third-party engine dependencies)
  • Cross-platform architecture
  • Future support planned for Vulkan, Metal, and DirectX
